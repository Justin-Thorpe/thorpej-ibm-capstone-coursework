# IBM Data Science Capstone – SpaceX Launch Analysis

This repository contains my work for the **IBM Data Science Capstone Project**, focused on analyzing and predicting outcomes of SpaceX rocket launches.

## Repository Structure

Each assignment is labeled by its module. Each file is a notebook, script, or dataset associated with a weekly objective.

## 📊 Project Overview

The goal of this project is to explore SpaceX launch data and build predictive models to determine the likelihood of a successful launch outcome.

Throughout the project, I worked through the full data science pipeline:

### Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing  
- Understanding launch success rates  
- Identifying key variables (payload, orbit, launch site, etc.)

### Web Scraping
- Extracted launch data from online sources  
- Structured and cleaned raw HTML data for analysis  

### Data Visualization
- Visualized trends using plots and maps  
- Analyzed launch site performance and geographic patterns  
- Explored relationships between payload, orbit, and success  

### Predictive Modeling
- Built classification models to predict launch success  
- Compared multiple algorithms (e.g., Logistic Regression, SVM, Decision Trees, KNN)  
- Evaluated model performance using accuracy and other metrics  

## Tools & Technologies

- **Python**
- **Pandas & NumPy**
- **Matplotlib & Seaborn**
- **Scikit-learn**
- **Folium** (geospatial visualization)
- **BeautifulSoup** (web scraping)
- **Jupyter Notebooks**

## Key Insights

- Launch success is influenced by payload mass, orbit type, and launch site  
- Certain launch sites show consistently higher success rates  
- Machine learning models can reasonably predict launch outcomes with tuned parameters
